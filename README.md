# SGBD-Projet-Ecommerce  
  
Projet académique de conception et d’implémentation d’une base de données relationnelle pour une plateforme e-commerce, couvrant la modélisation conceptuelle et logique, le dictionnaire des données, les scripts SQL, les vues métier et la gestion des rôles utilisateurs.  
  
Dans le cadre du cours SGBD à l'Université Paris Cité  
  
Enseignant : M. Alexis Ongagna  
  
Réalisé par : Yuan Ding & Minh Hang Do
